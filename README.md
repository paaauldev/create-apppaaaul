# Create AppPaaaul

Create projects as Paaul would.
Based on appcny

## Usage

```bash
# Interactive mode
pnpm create appncy

# Override template
pnpm create appncy --template <template>

# Override name
pnpm create appncy --name <name>

# Get Command help
pnpm create appncy --help
```

## Installation

```bash
pnpm install -g create-appncy
```
